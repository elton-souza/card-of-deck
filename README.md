## Card of Deck

### Para rodar o projeto é só seguir os passos:

- Clonar o projeto:

```sh
SSH: git@github.com:elton-souza/card-of-deck.git
HTTPS: https://github.com/elton-souza/card-of-deck.git
```

- Instalar as dependências do projeto:
```sh
cd card-of-deck
yarn ou npm install
```
- Iniciar o projeto:
```sh
yarn dev ou npm run dev
```
